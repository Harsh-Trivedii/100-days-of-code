Problem : Implementation of Queue using Circular Doubly Linked list

Explanation: Queue follows First In FIrst Out. In this question we are implementing Queue using Circular Doubly Linked list in which last node is connected
with first node. And each node has a previous and next pointer. In queue data is added at the rear end and deleted at front.

Sample Input Insertion: 1 2 3
Sample Output after deletion: 2 3

Use switch statement and let user select which operation he/she wants to perform [enqueue(), dequeue() and display()]
