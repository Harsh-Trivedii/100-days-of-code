Problem: Minimum indexed character
Question: Given a string str and another string patt. Find the minimum index of the character in str that is also present in patt.

Example:
Input:
str = adcffaet
patt = onkl
Output: -1
Explanation: There are none of the
characters which is common in patt
and str.

Expected Time Complexity: O(N).
Expected Auxiliary Space: O(Number of distinct characters).


Constraints:
1 ≤ |str|,|patt| ≤ 105 
'a' <= stri, patti <= 'z'

Test cases: 305 /305
