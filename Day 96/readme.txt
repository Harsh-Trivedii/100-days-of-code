Problem:Remove consonants from a string

Question:Given a string S, remove all consonants and print the modified string that contains vowels only.

Example 1:

Input
S = "abEkipo"
Output
aEio
Explanation : a, E, i, o are only vowels in the string.

Expected Time Complexity: O(|S|).
Expected Auxiliary Space: O(1).

Constraints
1 <= |S| <= 105
